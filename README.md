# Usage
<!-- usage -->
```sh-session
$ npm install -g kaput-cli
$ put COMMAND
running command...
$ put (-v|--version|version)
kaput-cli/0.0.0 darwin-x64 node-v12.17.0
$ put --help [COMMAND]
USAGE
  $ put COMMAND
...
```
